# Screenshots da loja

As capturas antigas (da marca anterior) foram removidas. Gere novas a partir do app DFNET:

1. Rode o app (APK de debug do GitHub Actions, ou `npm run dev` no navegador em modo celular).
2. Capture as telas: Login, Início, Boletos, Velocidade, Meus Apps, Suporte.
3. Formato recomendado pela Play Store: PNG/JPG, proporção de celular (ex.: 1080×1920),
   mínimo 2 imagens (recomendado 4–8).
